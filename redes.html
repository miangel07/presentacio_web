REDES: TIPOS DE REDES DE COMPUTADORAS red de área local (LAN) Una red de área local es una red que está limitada a un área geográfica pequeña, como una oficina, un edificio o una escuela. Esta red está gestionada por una única organización y se utiliza para compartir recursos como impresoras, archivos, etc. En esta red, los dispositivos se conectan a través de redes cableadas o inalámbricas. Las redes de área local pueden ser cableadas, lo que significa que los dispositivos se conectan mediante un cable Ethernet, o inalámbricas, que conectan los dispositivos a una red mediante tecnología Wi-Fi. Red de área amplia (WAN) Red de área amplia Un área geográfica que es más grande que una red local, como una ciudad, un país o incluso varios países. Estas redes se utilizan para conectar varias LAN y permitir la comunicación entre ellas. Las WAN están gestionadas por diferentes organizaciones y, a menudo, utilizan diferentes tecnologías de conexión, como líneas telefónicas, satélites, cables submarinos, etc. Red de Área Metropolitana (MAN) Una red de área metropolitana es una red que cubre un área geográfica más grande que una LAN pero más pequeña que una WAN. Esta red se utiliza para conectar diferentes LANs ubicadas en una misma ciudad o área metropolitana. Los dispositivos MAN están gestionados por diferentes organizaciones y, a menudo, utilizan tecnologías como fibra óptica, cableado Ethernet y Wi-Fi. Red de área de almacenamiento (SAN) Una red de área de almacenamiento se utiliza para conectar dispositivos de almacenamiento, como discos duros, a diferentes servidores. Esta red permite un acceso más rápido y eficiente a los datos almacenados. Las SAN utilizan tecnologías de conexión de alta velocidad, como la fibra óptica, para proporcionar un acceso rápido y seguro a los datos almacenados. CARACTERÍSTICAS DE LAS REDES DE COMPUTADORAS topología La topología de red se refiere a la forma en que se conectan los dispositivos. Las topologías más comunes son topologías en estrella, bus y anillo. protocolos Los protocolos son un conjunto de reglas que determinan cómo se comunican los dispositivos entre sí. Los protocolos más comunes son TCP/IP, HTTP, FTP y SMTP. Banda ancha El ancho de banda se refiere a la cantidad de datos que se pueden transmitir a través de una red por segundo. Más ancho de banda significa que la red puede transferir más datos en menos tiempo. seguridad La seguridad de los datos es una función importante de todas las redes informáticas. Se deben implementar medidas de seguridad para proteger los datos y los dispositivos contra posibles amenazas externas, como virus, piratas informáticos y malware.
