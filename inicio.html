hardware es el conjunto de componentes físicos a partir de los cuales se fabrica un dispositivo, y el software es el conjunto de programas o aplicaciones, instrucciones y reglas informáticas que permiten que el dispositivo funcione. Para este proyecto se necesitan algunos componentes hardware eficaz y productivo. Para poder Desarrollar una aplicación web, que lleve un paso a paso para la elaboración de distintos productos realizados a base de café, también se asignará una sección de compra y venta, donde el cliente pueda ver más detalladamente los productos y así poder adquirir los ya mencionados. Esto también es un gran avance para las personas innovadoras y apasionadas por el tema de la gastronomía y el café.
